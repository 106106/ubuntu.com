• Set up and install a vm using Multipass

    1. Make sure the network you’re connected to is marked Private, otherwise Windows will prevent Multipass from starting.

    2. Download Multipass for macOS

    3. Activate the downloaded installer and it will guide you through the steps necessary. You will need an account with Administrator privileges to complete the installation.

      {{ image ( url="https://assets.ubuntu.com/v1/ac6f638d-multipass-macos-install-screen.png", alt="", width="658", height="475", hi_def=True, loading="lazy", ) | safe}}

    4. There’s a script to uninstall multipass if you need it: 

      sudo sh "/Library/Application Support/com.canonical.multipass/uninstall.sh"
