• Set up and install a vm using Multipass

    1. Install Multipass: 

      sudo snap install multipass
      Note:

      Multipass is about to name your appliance something like ‘cute-emperor’ or ‘prophetic-giraffe’. After this whatever you get will be your <appliance name>. To use a different name simply add '--name <your new name>' to the end of the next line.
