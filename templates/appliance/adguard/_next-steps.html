Start using your {{ appliance.name }} Ubuntu Appliance

When you run AdGuard Home for the first time, it will start listening to http://0.0.0.0:3000 or http://127.0.0.1:3000 and prompt you to open it in your browser.

There you will go through the initial configuration wizard.

Now you can follow the instructions on the screen to get started using your new {{ appliance.name }} Ubuntu Appliance. For more information or more documentation on what to do next, the AdGuard website is the best place to go next.
