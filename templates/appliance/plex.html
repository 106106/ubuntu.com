Plex Media Server Ubuntu Appliance

By Plex, Inc.

Plex Media allows you to organise and stream your own collection of movies, TV, music, podcasts and more from one place. Gone are the days of frustration when your favourite shows aren’t on Netflix or when someone else is using your Spotify or a Prime account.

Install Plex on Raspberry Pi

What you will need

  • A microSD card (4GB minimum, 8GB recommended)
  • A computer with a microSD card drive
  • A Raspberry Pi 2, 3 or 4
  • A micro-USB power cable (USB-C for the Pi 4)
  • A Wi-Fi network or an ethernet cable with an internet connection
  • A monitor with an HDMI interface
  • An HDMI cable for the Pi 2 & 3 (micro HDMI cable for the Pi 4)
  • A USB keyboard

Get set up

  1. Download the Plex Ubuntu appliance image

  2. Use the Raspberry Pi imager to flash to an SDcard

  3. Setup Ubuntu Core

  4. On a computer or mobile device, visit your Pis IP in the browser at port 32400. (E.g http://192.168.1.107:32400)

  5. That’s it!

Install Plex on KVM

What you will need

  • An Ubuntu SSO account with an SSH key
  • A computer running Ubuntu Desktop

Get set up

  1. Start by creating an Ubuntu SSO account

  2. Import an SSH Key into your Ubuntu SSO account (instructions).

  3. Download the Ubuntu Core image for amd64

  4. Uncompress the image with the following command: unxz ubuntu-core-16-amd64.img.xz

  5. Install the qemu-kvm package with the following command: sudo apt install qemu-kvm

  6. Then, run the kvm-ok command to check KVM status and your hardware: kvm-ok

  7. The message should say: INFO: /dev/kvm exists KVM acceleration can be used

  8. You can now launch a virtual machine with KVM, using the following command:

    kvm -smp 2 -m 1500 -netdev user,id=mynet0,hostfwd=tcp::8022-:22,hostfwd=tcp::8090-:80 -device virtio-net-pci,netdev=mynet0 -vga qxl -drive file=ubuntu-core-16-amd64.img,format=raw

    Note:this command sets up port redirections:
    localhost:8022 is redirecting to port 22 of the virtual machine for accessing it through SSH
    localhost:8090 is redirecting to its port 80

  9. You should now see a window, with your Ubuntu Core virtual machine booting inside it.

  10. The system will boot then become ready to configure. The device will display the prompt “Press enter to configure”

  11. Press enter then select “Start” to begin configuring your network and an administrator account. Follow the instructions on the screen, you will be asked to configure your network and enter your Ubuntu SSO credentials

  12. At the end of the process, you will see your credentials to access your Ubuntu Core machine: This device is registered to [Ubuntu SSO email address]. Remote access was enabled via authentication with the SSO user [Ubuntu SSO user name] Public SSH keys were added to the device for remote access Once setup is done, you can login with SSH into Ubuntu Core, using the following command: ssh -p 8022 [Ubuntu SSO user name]@localhost

Your user name is your Ubuntu SSO user name, which you will have been reminded of at the end of the account configuration step.
