Security first

Crisp. Clean. Core.

We believe that security is everything. That’s why Ubuntu Appliances meet state-of-the-art standards of endpoint security.

Security capabilities like strict software isolation and software immutability are provided out-of-the-box with each appliance, thanks to Ubuntu Core.

To keep your device secure, CVE patches and bug fixes will be delivered over the long term and in a timely manner, the Canonical way.

Always fresh

Self-healing software

Keeping up with upstream, automatically!

Whether it is stable, candidate or beta releases that you want to run on your box, the system will autonomously fetch and install software updates.

What’s more, failed updates will be reversed without your intervention.
