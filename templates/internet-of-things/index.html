Ubuntu for the Internet of Things

Ubuntu Core, also known as Snappy, is a stripped down version of Ubuntu, designed to run securely on autonomous machines, devices and other internet-connected digital things. From homes to drones, these devices are set to revolutionise many aspects of our lives, but they need an operating system that is different from that of traditional PCs.

What is a Ubuntu Core?

Ubuntu Core is the smallest Ubuntu ever — a tiny, transactional operating system that can run on everything from IoT devices to large-scale container deployments. And with it comes a new kind of app: the super-secure, ultra-reliable ’snap’.

What is a snap?

A snap is a secure, easily upgraded and distributed, universal Linux app packaging format, which contains a compressed app and all its dependencies.

Snap packages are:

  • Confined for strict security
  • Transactional for reliable updates
  • Compressed to save disk space
  • Widely adopted by industry leaders

Why Ubuntu Core?

Create differentiated devices that will continue to generate revenue over your product’s lifetime, with a software platform that’s easy to update and upgrade.

  • It is fast and reliable with strong security features for apps and users
  • Atomic, transactional upgrades and rollbacks for both apps and Ubuntu Core itself make it simple and safe to maintain the software on your remote devices
  • Both the OS and applications are separated into distinct read-only images, making it safe to have multiple apps on a single device
  • The snap format is a better, simpler application packaging system
  • Signature authentication ensures what’s running is exactly what developers have published
