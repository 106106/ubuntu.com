Get OpenStack Autopilot to run an Ubuntu cloud

Canonical’s OpenStack Autopilot is a fully automated deployment of an OpenStack cloud — just add servers.

Autopilot is the world’s easiest dynamic OpenStack installer. It handles every aspect of your cloud both during installation, expansion, and everyday operations. To start, you will need at least five machines with two disks each. Two machines need to have two network interfaces (NICs).
