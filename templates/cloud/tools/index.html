Cloud tools

Want to get your cloud infrastructure up and running fast? We’ve got all the tools you need to deploy, scale and manage an OpenStack cloud, quickly and cost‐effectively.

Modelling with Juju

Juju is a service orchestration tool that enables you to quickly model, configure, deploy and manage entire cloud environments with only a few commands. Use it to deploy OpenStack, your own code or any of the 300 services and pre‐configured applications available in the Juju store.

Read more about Juju ›

Deploy, monitor and manage your cloud with Landscape

Landscape is where the power of Ubuntu OpenStack really comes together. It allows you to build your cloud in minutes, then monitor it, manage it and produce timely and accurate compliance reports. The browser‐based GUI gives you an instant overview of everything going on in your cloud, from hardware to workloads.

Learn more about Landscape ›

Provisioning with MAAS

MAAS is a bare–metal provisioning tool that takes the pain out of detection and configuration and gets your servers ready for Juju. You can think of it as turning your hardware into a cloud.

Read more about MAAS ›

LXD — the Linux container hypervisor

LXD is the next generation hypervisor from Canonical. It has the density of containers with manageability of virtual machines. Launch a new Linux machine in under a second. Launch hundreds of them on a single server. Connect them separately and securely. Run that on one node or a million. On bare metal, public cloud, or OpenStack.

Read more about LXD ›

Snappy Ubuntu Core

Snappy is a new way to use Ubuntu in the cloud with transactional, image-based updates. Snappy Ubuntu Core is the same OS with the same libraries as today’s Ubuntu, but applications are provided through a simpler, faster mechanism and, most importantly, it provides stronger security guarantee for apps, making it ideal for Docker and other cloud deployment frameworks.

Read more about snappy ›
