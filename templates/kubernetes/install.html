Install Kubernetes yourself

Canonical has developed multiple installation tools to get you up and running with Kubernetes based on your needs.

Three ways to install Kubernetes

Try it locally

For a simple all-in-one installation on your local machine in under 60 seconds, try microk8s.

Deploy to your cluster

For a guided cluster installation on any cloud or data center from the CLI, try conjure-up.

Install with Juju

Try Juju’s drag and drop, browser based tool or for a detailed walk through of a CLI-based installation, go to Juju charm bundle for Kubernetes.

Getting Started with The Canonical Distribution of Kubernetes

Install on a public cloud

There are two ways to install Kubernetes in your software defined infrastructure on any major public cloud with conjure-up or Juju.

  • Install Kubernetes with conjure-up
  • Install and manage Kubernetes with Juju

Install on your local machine

Canonical has developed microk8s, a simple, streamlined mechanism to get Kubernetes setup on any desktop. Microk8s is quick to install and has a small footprint suitable for local development and testing.

  • Visit microk8s.io
  • Find the project on github

Install on a physical cluster

Deploy on an OpenStack cloud using MAAS to provision the physical servers and conjure-up to install OpenStack and Kubernetes.

  • Use conjure-up to install OpenStack or Kubernetes
  • Find our more about MAAS
