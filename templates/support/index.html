Enterprise Linux and open source support

Canonical provides 24/7, enterprise-grade support for the full stack through Ubuntu Advantage. Customers can choose between two support offerings - Ubuntu Advantage for Applications and Ubuntu Advantage for Infrastructure. See which offering is best for your use case to ensure your production stack is supported.

Ubuntu Advantage

Applications

Security and support for open source database, LMA, server and cloud-native apps.

  • Security patching for over 28,000 packages in Ubuntu Universe and Main repositories
  • Up to 10 years support* for Ubuntu LTS
  • Technical support for open source database, LMA and cloud-native apps

Supported applications ›

Infrastructure

Security and IAAS support for open source infrastructure.


  • Security patching for more than 2,500 packages in Ubuntu Main repository (after 5 years of free LTS security patching)
  • Up to 10 years support* for Ubuntu LTS
  • Technical support for OpenStack, KVM, MAAS

Supported infrastructure ›

*Ubuntu 18.04 LTS onwards

The most comprehensive Linux enterprise subscription

  • 24/7 phone support

    Direct 24/7 access to our world-class enterprise open source support team through our web portal, knowledge base or via phone.

  • Support for OpenStack, K8s, LXD, MAAS and Ceph

    A single subscription for security and support of Ubuntu and a range of infrastructure.

  • Kafka, NGINX, MongoDB and PostgreSQL support

Systems management and security at scale

Automate day-to-day management with Landscape, the most cost-effective tool to support and monitor large and rapidly expanding networks of clouds, servers and desktops. Use Landscape for audit reporting, security compliance and patch management at scale. Landscape is available as a service from the cloud or on-premises for compliance with stricter security requirements.

  • Manage physical, virtual and cloud-based systems
  • Automate package management
  • Maintain security and compliance
  • Control inventory
  • API access for integration and reporting
  • Role-Based Access Control
