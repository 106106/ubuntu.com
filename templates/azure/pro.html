Ubuntu Pro for Azure

For production. For professionals.

Ubuntu Pro for Azure is a premium image designed by Canonical to provide the most comprehensive feature set for production environments running on Azure public cloud. Offered as a virtual machine image through Azure Marketplace, it includes a subscription to Canonical's critical security and compliance services by default. Ubuntu Pro is suitable for small to large-scale Linux enterprise operations — no contract necessary.

What’s in Ubuntu Pro?

  • All the Ubuntu goodness

    Ubuntu Pro is based on Ubuntu and leverages both the breadth of open source included and the Azure optimisations built in.

  • Kernel Livepatch

    Ubuntu kernel updates are regularly issued. With our Livepatch service, kernel patches are delivered immediately, without the need to reboot your VMs.

  • Open source security

    Ubuntu Pro adds security coverage for the most important open source applications like Apache Kafka, NGINX, MongoDB, Redis and PostgreSQL.

  • FIPS 140-2 and Common Criteria (CC) EAL2

    Officially certified components to enable operating environments under compliance regimes like FedRAMP, HIPAA, PCI and ISO.

  • Hardening with CIS and DISA STIG profiles

    For companies looking to leverage industry benchmarks for hardening, Ubuntu Pro makes two leading profiles available out of the box.

  • Fully Azure-integrated

    Starting in Q2 2020, Ubuntu Pro will include integrations with Azure services including Azure Security Center, Azure Monitor and Azure Log Analytics.

  • The right Ubuntu for you

    Canonical brings the three most popular Ubuntu Server distributions to choose from - 14.04 LTS, 16.04 LTS, and 18.04 LTS.

  • 10-year lifetime

    Canonical backs Ubuntu Pro for 10 years*, ensuring security updates are available throughout, with a guaranteed upgrade path.

  • Priced for the cloud

    Ubuntu Pro pricing tracks the underlying compute cost, starting at under $0.01 for the smallest B1ls instance type, and ramping down to less than 1% of hourly compute.

* For 18.04 LTS onwards. The maintenance period for Ubuntu Pro 14.04 LTS and 16.04 LTS is 8 years.

Broadest open source security coverage keeps CISOs sleeping at night

Ubuntu Pro includes security coverage for the entire collection of software packages shipped with Ubuntu, tracking high and critical CVEs. This makes it ideal for companies that have embraced open source for their new products and need to ensure security for production and mission-critical workloads.

One key reason why developers prefer Ubuntu is that it includes a huge set — in total, over 26,000 packages — of upstream open source known as ‘universe’. Universe includes workloads like NGINX, MongoDB, Redis, Kafka, but also language ecosystems like Python, Java, Ruby and Javascript/Node.js. With Ubuntu Pro's built-in CVE tracking and patching, CIOs and security officers can be confident they can roll these packages out to production.
